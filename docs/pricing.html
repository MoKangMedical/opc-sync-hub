常见问题

关于定价和方案的常见疑问

可以随时升级或降级方案吗？

可以。您可以随时在账户设置中升级或降级方案。升级立即生效，降级将在当前计费周期结束后生效。升级时我们会按比例计算差价。

免费方案有什么限制？

Starter 方案完全免费，支持最多 5 个仓库的基础同步功能和每日健康检查。它不包含 CI/CD 编排、依赖管理和发布协调等高级功能。适合个人项目和初步体验。

14 天试用期结束后会自动扣费吗？

不会。试用期结束前我们会发送提醒邮件。如果您不主动选择付费方案，账户将自动降级为 Starter 免费方案，您的数据不会丢失。

支持哪些支付方式？

支持支付宝、微信支付、银行卡转账以及国际信用卡（Visa/Mastercard）。企业客户支持对公转账和合同付款。

私有化部署需要什么条件？

Enterprise 方案支持私有化部署。您需要提供至少 4 核 CPU / 8GB 内存的服务器（支持 Docker/K8s）。我们的工程师会协助您完成部署和配置。

开源项目有优惠吗？

有的！我们为开源项目提供 Professional 方案的免费使用权。请通过 GitHub Issues 提交申请，附上项目链接和简要说明。
